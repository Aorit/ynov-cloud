### Installation

- Install dependencies
  ```bash
  npm install
  ```

- Copy `.env.local.example` file to create your `.env` file


- Run app
  ```bash
  npm run dev
  ```
  
### Test

- Launch all tests
  ```bash
  npm run test 
  ```
  
- Run coverage
  ```bash
  npm run test:coverage
  ```

### Build

- Build app
  ```bash
  npm run build
  ```
